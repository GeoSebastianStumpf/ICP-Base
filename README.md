## Welcome to ICP-Base

ICP-Base is a free and open-source software for processing and visualizing Laser Ablation–Inductively Coupled Plasma–Mass Spectrometry (LA-ICP-MS) data. It provides a transparent and flexible environment for quantifying mineral spot analyses, fully enclosed solid and fluid inclusions, and line scan measurements. ICP-Base integrates raw data, metadata, intermediate processing steps, and final results within a centralized hierarchical database, preserving the full analytical context of each measurement. Through built-in visualization based on the VisModule from MinPlotX, users can explore data and generate publication-ready figures directly within the software. Designed with a modular architecture and fully accessible source code, ICP-Base supports reproducible workflows and enables community-driven development for future LA-ICP-MS data processing methods.

This software was developed by Sebastian Stumpf in close collaboration with Nils B. Gies, Thomas Pettke, and Marcel Guillong, whose contributions and expertise helped shape the design and functionality of the software.
